## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* PHP
* Visual Studio Code(Optional)

### Installation

1. Clone the repo
```sh
git clone https://github.com/HridayHegde/Cake-Your-Way.git
```
2. Run using Cmd or via Visual Studio Code
